上下文即资产

上下文，Context，将会是AI 时代每个人最重要的资产

在人工智能的通用能力变得越来越强的大环境下
应用层的产品或服务永远会碰到一个问题——理解用户

在过去，理解用户需要用 AB Test 需要做调研做访谈
通过大量的数据通过推荐引擎来发现用户的需求
但是这个需求是隐形的 统计量的
不是聚焦的 个体的

所以任何一个产品，Before AI，没办法做到完全的需求定制
没办法彻底的做到挖掘用户的全生命周期价值
只能在总体的统计量上做到取舍与平衡

但是有了上下文，Context 之后，相当于用户的方方面面，当然包括需求
被用自然语言这种纯透明的方式暴露给了 AI 产品，某一个 AI 产品——因为我们常用的 AI 产品就只有一个
个人而言，是 ChatGPT
但是除了更好的单一产品的使用体验外，我没有获得任何其他的东西
即便他可以产生更多我们现在甚至未曾能想象的价值
比如钱 比如更多个性化定制的服务 等等

如果可以把 Context 沉淀为用户所拥有的价值
用户自己拥有自己的 Context 这里对比的是传统互联网（Web2）数据由大公司垄断
可以通过自己的 Context 创建收入
例如部署 Agent 为他人提供服务 获得收入 例如注入某些产品 以获得更好的个性化的体验
由例如帮助用户“延长寿命” 因为 Context 就代表这个人的思维
而 Agent 未来必然可以在更大的范围内“行动”
这就是 AI 时代的核心 AI 时代的操作系统 AI 时代的唯一重要的资产

想象一下
1. Context 可以在你睡觉的时候，为你挣钱
2. Context 可以注入到你的代码编辑器中，让喜欢写驼峰命名的你不再被 AI 生成的下划线所困扰
3. Context 可以注入到你的邮箱中，作为一个筛选器，筛选出真正与你有关且你在乎的邮件，整理成你所喜欢的 bullet 风格供你早上快速查阅
4. Context 可以注入到你的恋爱 Agent 分身中，帮你留意可能的那个灵魂契合的伴侣，然后尝试与他产生联系
5. Context 可以注入到你的工作 Agent 分身中，在老板布置了简单的紧急任务时，先自己处理，然后在通知正在度假的你

…

一切都来自于 Context，独属于你的 Context

进一步的，宏观来说
无数个体的 Context 组合在一起的网络 就是 Web3 中构想了许久却无人能够实现的价值互联网
人们不但可以读信息 写信息 像小红书一样 而且可以直接拥有信息 所有因信息产生的价值都直接分配

这就是未来的互联网
由 Context，AI 和 人组成的，新的互联网
谁能用最合适的方式垄断Context，谁就将成为下一个 microsoft，拥有未来互联网的定义权
